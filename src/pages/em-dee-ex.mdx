### Syntax highlighting

This is roughly the snippet that generated the GraphQL output above:

```jsx
render () {
  return (
    <blockquote>
      <code>"{props.data.site.siteMetadata.description}"</code>
    </blockquote>
  )
}

export const pageQuery = graphql`
  query {
    site {
      siteMetadata {
        description
      }
    }
  }

`
```
